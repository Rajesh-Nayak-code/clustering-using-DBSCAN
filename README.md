# Moon Dataset Clustering: K-Means vs DBSCAN

## Overview

This project demonstrates **unsupervised learning** by applying and comparing **K-Means** and **DBSCAN** clustering algorithms on the Moon Dataset generated using Scikit-Learn.

The project highlights how different clustering algorithms perform on non-linear, moon-shaped data.

---

### Characteristics of DBSCAN
- Does not require the number of clusters.
- Groups points based on density.
- Can detect outliers automatically.
- Handles arbitrarily shaped clusters.

---

## Results

### K-Means
- Divides data based on distance from centroids.
- Struggles with non-linear cluster shapes.

### DBSCAN
- Correctly identifies the two moon-shaped clusters.
- Performs better on irregularly shaped data.
- Can label noise points as outliers.

---
